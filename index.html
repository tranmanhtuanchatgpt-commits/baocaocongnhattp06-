🔥 Kết nối Firebase – Đồng bộ thời gian thực
⚫ Chưa kết nối – đang dùng dữ liệu cục bộ trên máy này
📌 Hướng dẫn thiết lập Firebase (làm 1 lần duy nhất):
  1. Vào console.firebase.google.com → chọn project của bạn
  2. Menu trái: Build → Realtime Database → Create database
  3. Chọn vùng Singapore → Start in test mode → Enable
  4. Sau khi tạo xong, copy Database URL (dạng: https://TÊN-default-rtdb.asia-southeast1.firebasedatabase.app)
  5. Dán URL vào ô bên dưới → Bấm Kết nối & Lưu
⚠️ Quy tắc bảo mật: Trong Firebase Console → Realtime Database → Rules → đảm bảo có: "read": true, "write": true
